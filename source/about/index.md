由于开源软件下载资源一般都在 GitHub Releases 页面，下载速度极慢。

有国内镜像的，直接提供国内镜像下载地址，没有国内镜像的，如果小于25MB，将下载好的软件放在仓库中，通过 jsdelivr 提供的 CDN 进行下载，大于25MB的再想办法吧。
